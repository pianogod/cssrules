Specificity

Ranking Definition Examples
1. HTML Inline Style
2. ID Attribute
3. Class, Attribute, Pseudo-class
4. Type (element), Pseudo-element 		A higher specificity score in any category outweighs any number of lower specificity scores. For example, a single ID selector (0,1,0,0) is more specific than any number of class selectors (0,0,n,0).

In the case of a specificity tie, the last rule defined in the CSS is the one that is applied. 		#navbar has a specificity of (0,1,0,0) (one ID).
.button.active has a specificity of (0,0,2,0) (two classes).
div span has a specificity of (0,0,0,2) (two elements).
#navbar ul li a has a specificity of (0,1,0,3) (one ID and three elements).
